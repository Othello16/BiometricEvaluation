FRVT 1:N Target Brief
=====================

Official public source:

 * `https://github.com/usnistgov/frvt`

Target model:

 * face identification target
 * vendor must conform to the official FRVT 1:N package for API and validation

Jupiter tasks:

 * keep gallery-search semantics inside the native target package
 * avoid wrapping the interface behind a REST-only abstraction
 * pin runtime-library dependencies before envelope packaging
