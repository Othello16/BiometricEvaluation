FRVT 1:1 Target Brief
=====================

Official public source:

 * `https://github.com/usnistgov/frvt`

Target model:

 * face verification target
 * vendor must conform to the official FRVT 1:1 package for API and validation

Jupiter tasks:

 * fetch the exact FRVT 1:1 package from the upstream repo
 * map vendor deliverables to that package without adding a substitute API
 * validate natively first
 * only then build a hardened runtime envelope
